# Great Britain regional boundaries

`gb-regions.topo.json` is the ONSVisuals `geogGBregion.json` super-generalised boundary file.

- Source: https://github.com/ONSvisual/topojson_boundaries
- Original data: ONS Open Geography Portal
- Coverage: Scotland, Wales, and the nine English regions
- Licence notice: Contains Ordnance Survey, Office for National Statistics, and National Records Scotland data © Crown copyright and database right. Ordnance Survey data is covered by the OS OpenData Licence.
